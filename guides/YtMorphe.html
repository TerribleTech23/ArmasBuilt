YouTube Morphe

A specialized, modified YouTube client built for the Android ecosystem. Ad-free viewing, background playback, and SponsorBlock integration.

Security Notice: Always download these tools from official sources. This guide points directly to the MorpheApp organization to ensure legitimate APKs.
Step 01

Set Up microG-RE

To sign in to your Google account without using the official YouTube app, you need a compatibility layer. Morphe works best with the microG-RE fork.

Download microG-RE
Step 02

Install via Obtainium

Add the Morphe repository to Obtainium to track and update the app directly from GitHub.

  • Open Obtainium and click "Add App".
  • Paste: https://github.com/MorpheApp/YouTubeMorphine
MorpheApp GitHub
Step 03

Optimization Settings

Navigate to Settings > Morphe Settings to enable the core features:

  • SponsorBlock: Auto-skips sponsor segments.
  • Return YouTube Dislike: Re-enables the dislike counter.
  • OLED Theme: Pure black theme for battery savings.
Step 04

Disabling the Official App

Ensure links open in Morphe by default by disabling the stock YouTube app via Settings > Apps.
